### single_run
#### Folder "c++"
c++ codes for simulation. 

#### Folder others
R and python code for visualization.

`reduce_dim.py`: 
Applying PCA to summarize the phenotype distribution.
For phenotype calculation, please set the correct parameters. 

`plot.R`:
Plot the proportion of male-like and female-like individuals (Figure 2B).

`plot_pca.R`:
Plot the developmental trajectories at a specified time (Figure 2C).
The variable "index" denotes the locus index whose genotype determines the color. 

`plot_association.R`:
Plot pvalues of Kendall's tau-b test (Figure 2D). 

`plot_geno_pheno.R`:
Plot genotypes and phenotypes of subset of individuals (Extended Data Figure 2).
Plotted genotypes are chosen from most frequent ones.
